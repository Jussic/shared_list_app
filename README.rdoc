== README

LIST

This is for those who wish to envision everything they need to do, and create visuals of tasks that need t be perfomed. Welcome to "Lists", where you can create running lists of things that need to be done, and prioritize your lists .

A web app that creates lists, and coordinates with your calendar!

This project is developed as a part of the mentorship program "Learn IT, Girl!". Special thanks goes out to Anahita, appreciate your supervision and mentorship!
